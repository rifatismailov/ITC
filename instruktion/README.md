## ⚡ Швидка шпаргалка

### 🔐 Крок 1 — OpenVPN

| Параметр | Значення |
|----------|----------|
| Логін | `student_##` *(замініть ## на ваш номер)* |
| Пароль | `student_Hysg&63` |
| Сертифікат | отримати у викладача або завантажити з платформи |

**Кроки:**
1. Встановіть [OpenVPN Connect](https://openvpn.net/client/)
2. Імпортуйте файл `student_##.ovpn`
3. Введіть логін та пароль → натисніть **Connect**
4. Статус змінився на `Connected ✓` — можна переходити далі

### 🖥️ Крок 2 — Proxmox (тільки після VPN!)

| Параметр | Значення |
|----------|----------|
| URL | `https://192.168.88.88:8006` |
| Логін | `student_##` *(ваш номер)* |
| Пароль | `Student@` |
| Realm | `Proxmox VE authentication server` |

**Кроки:**
1. Відкрийте браузер → перейдіть на `https://192.168.88.88:8006`
2. Якщо браузер попередить про сертифікат — натисніть **"Додатково" → "Перейти до сайту"** *(це безпечно)*
3. Введіть логін та пароль → натисніть **Login**

## 🚀 Як увімкнути GitHub Pages

> Щоб повна HTML-інструкція була доступна за посиланням:

1. Відкрийте репозиторій на GitHub
2. Перейдіть у **Settings** → **Pages**
3. У розділі **Branch** оберіть `main` та папку `/ (root)`
4. Натисніть **Save**
5. Через 1-2 хвилини сторінка буде доступна за адресою:  
   `https://YOUR_USERNAME.github.io/YOUR_REPO/student_guide.html`
6. Оновіть посилання у верхній частині цього README
